# DBSS Knowledge Platform

## Overview

DBSS Knowledge Platform is an AI-powered enterprise knowledge management platform.

The platform extracts information from enterprise systems such as:

- Confluence
- Jira
- Swagger/OpenAPI
- Git repositories

It builds a searchable knowledge base and automatically generates:

- API Documentation
- Configuration Documentation
- Troubleshooting Guides
